##

A Minecraft datapack that adds alternative vanilla-friendly crafting recipes, allowing players to use logs directly in any recipe that normally requires only wooden planks — **save time**!

<details>
## <summary>📖 Recipes</summary>

- Sticks from logs/stems/blocks-of-bamboo
- Chests from logs/stems/blocks-of-bamboo
- Wooden stairs from logs/stems/blocks-of-bamboo
- Wooden slabs from logs/stems/blocks-of-bamboo
- Wooden pressure plates from logs/stems/blocks-of-bamboo
- Wooden doors from logs/stems/blocks-of-bamboo
- Wooden trapdoors from logs/stems/blocks-of-bamboo
- Wooden boats from logs
- Bamboo rafts from blocks of bamboo
</details>

## 📄 Installation

1. Download the datapack `.zip` file
2. Place it in your world's `datapacks/` folder
3. Run `/reload` in-game or load a new world
